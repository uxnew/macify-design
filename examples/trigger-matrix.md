# Trigger Matrix

Macify Design triggers by workflow verbs, not by industry nouns.

| Request | Trigger? | Reason |
| --- | --- | --- |
| Build a tutorial system HTML to manage and publish tutorials | Yes | Publishing manager and content operations tool |
| Build a tutorial website for reading HTML/CSS lessons | No by default | Passive reading site |
| Build a documentation CMS with review workflow | Yes | Manage, edit, publish, review |
| Build a static documentation site | No by default | Passive reading |
| Build an AI prompt manager | Yes | Manage, generate, preview, organize |
| Build an AI product landing page | No by default | Marketing |
| Build an order management admin | Yes | Manage operational objects |
| Build an ecommerce storefront | No by default | Storefront, not admin tool |
| Build a video asset review tool | Yes | Review, manage, publish media |
| Build a video portfolio page | No by default | Showcase |
| Build a monitoring dashboard | Yes | Monitor and analyze |
| Build a data storytelling landing page | No by default | Presentation, not workflow |

Explicit user requests for Mac-like design can override the default, but the result must still avoid macOS simulation and Apple assets.
