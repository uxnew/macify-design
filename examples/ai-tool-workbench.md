# Example: AI Tool Workbench

Prompt:

> Build an AI prompt management tool.

Macify Design should trigger.

Expected product interpretation:

- Product type: AI workbench.
- Core objects: prompts, folders, runs, model settings, outputs.
- Primary workflow: create prompts, organize, test, compare outputs, save versions.

Expected desktop layout:

- Sidebar: folders, recent prompts, tags.
- Toolbar: search, new prompt, run, compare, export.
- Main workspace: prompt editor, output preview, run history.
- Inspector: model, temperature, variables, metadata, version info.
- Status: queued, running, saved, failed, token estimate.

Expected mobile layout:

- Prompt list and search first.
- Prompt editor as main view.
- Model/settings in bottom sheet.
- Run output in drill-in or tab.
- Advanced comparison can be desktop-best, but running and viewing remain usable.
