# Counterexample: Knowledge Base Reading Site

Prompt:

> Build a knowledge base website where users read articles and browse categories.

Macify Design should not trigger by default.

Reason:

- Primary workflow is reading and browsing.
- The product is a passive content experience, not an operational tool.
- A windowed workbench would likely harm content readability and site intent.

Possible exception:

If the user asks for a knowledge base management system where admins create, edit, tag, review, publish, and analyze articles, Macify Design should trigger.
