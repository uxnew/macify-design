# Apple Boundary Rules

Macify Design is unofficial and not affiliated with Apple. Use Mac-like product design ideas without copying Apple products or assets.

## Allowed

- Windowed workbench structure.
- Sidebar, toolbar, split view, inspector, sheet, popover, and status feedback patterns.
- Light/dark appearance.
- Semantic color and material layering.
- System font stack such as `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`.
- Project-approved icon libraries.
- General ideas of compact controls, separators, subtle shadows, and precise states.

## Not Allowed

- Apple logo.
- Apple product screenshots.
- Dock.
- System menu bar.
- Fake macOS desktop.
- Fake operating-system window manager.
- Finder clone.
- System Settings clone.
- Mail/Notes/Calendar clone.
- Traffic-light dots used as functional product controls.
- SF Symbols by default unless the user's project explicitly and legally uses them.
- Claims that the result is official macOS UI.

## Language

Prefer:

- Mac-like.
- macOS-inspired.
- Apple-computer-like product skin.
- Windowed web workbench.

Avoid:

- macOS clone.
- Apple official UI.
- Exact Finder/Settings/Mail style.

## Validation

Fail the design if a user could reasonably think it is pretending to be the operating system rather than a web product inspired by Mac-like skin and layout.
